# Id-Yuu.github.io
Simple Portfolio with vanilla js

## Install
Clone this repo
```
git clone https://github.com/Id-Yuu/Id-Yuu.github.io.git
```
Open folder
```
cd Id-Yuu.github.io
```
install module
```
npm install
```
run local
```
npm run dev
```

- Vanilla Js
